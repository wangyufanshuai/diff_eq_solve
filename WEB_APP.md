# 科学计算智能体 Web 界面

在项目根目录运行：

```powershell
streamlit run diff_eq_solver/agent_web.py
```

界面支持输入自然语言问题、选择数学物理方程模板、编辑参数和初值/边界条件 JSON、选择精度、运行求解，并在中文标签页中查看：

- 总览
- 推导
- 解
- 图像
- 误差
- 代码
- 数据
- 警告
- 文献

求解策略是“符号解析优先，数值求解兜底”。如果问题缺少必要的初值或边界条件，系统会给出中文警告；如使用演示默认值，也会明确标注该结果不是用户指定条件下的唯一精确结论。

注意：本项目覆盖教材核心数学物理方程族和常见 ODE/PDE 模板，不承诺任意复杂方程都存在闭式解析解。

## 教材案例

Web 界面提供教材案例预设，包括：

- 热方程
- 波方程
- Poisson 方程
- Schrodinger 本征态
- Maxwell 一维波
- 浅水方程

这些案例覆盖解析优先、数值兜底、本征值/BVP、PDE 系统模板和误差/稳定性报告。复杂几何、三维湍流 DNS、任意多物理耦合不属于当前轻量版本的直接求解范围。
